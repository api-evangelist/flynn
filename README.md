# FlynnThis is a repo for managing machine readable artifacts for the [Flynn APIs](https://flynn.io/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Flynn APIs](https://flynn.io/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://flynn.io/)- [Developer](https://flynn.io/)- [GitHub](https://github.com/flynn)The [APIs.json](https://github.com/api-evangelist/flynn/blob/master/apis.json) index provides links to all the relevant links for the Flynn APIs, including the OpenAPI and Postman if they are present.
